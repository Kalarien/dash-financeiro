🎯 GUIA: COMO USAR O CLAUDE NO NOTEBOOK PARA ATUALIZAR O DASHBOARD
===============================================================================

Quando você estiver no notebook e precisar atualizar o dashboard, siga este guia:

===============================================================================
PASSO 1: PREPARAR OS ARQUIVOS NO NOTEBOOK
===============================================================================

Você precisa ter estes arquivos no notebook (pode baixar do GitHub ou copiar deste PC):

✅ OBRIGATÓRIOS (para o Claude entender):
   1. CONTEXTO_PARA_CLAUDE.txt          (este arquivo é o prompt inicial)
   2. RESUMO_TECNICO.txt                (resumo técnico completo)
   3. TUTORIAL_ATUALIZACAO_MENSAL.md    (tutorial passo a passo)
   4. CHECKLIST_ATUALIZACAO.md          (checklist de atualização)

⚠️ OPCIONAIS (úteis mas não essenciais):
   5. auxiliar_atualizacao.py           (script auxiliar)
   6. modulo_fechamento_mes.py          (para o Claude ver o código atual)
   7. modulo_fluxo_caixa.py             (para o Claude ver o código atual)

💡 DICA: Clone o repositório no notebook e você terá TODOS os arquivos!
   git clone https://github.com/Kalarien/dash-financeiro.git

===============================================================================
PASSO 2: COLETAR OS DADOS FINANCEIROS
===============================================================================

ANTES de falar com o Claude, colete todos os dados do mês:

□ Acessar Asaas e anotar: faturamento, saldo em conta, a receber
□ Acessar Pagar.me e anotar: faturamento, saldo em conta, a receber
□ Acessar Stripe e anotar: faturamento, saldo em conta, a receber
□ Acessar Crypto e anotar: faturamento, saldo em conta, a receber
□ Verificar vendas B2B e valores a receber
□ Calcular faturamento total
□ Calcular total de taxas pagas

Use o CHECKLIST_ATUALIZACAO.md para não esquecer nada!

===============================================================================
PASSO 3: ABRIR O CLAUDE NO NOTEBOOK
===============================================================================

1. Abra o Claude Code ou Claude.ai no notebook
2. Inicie uma nova conversa

===============================================================================

===============================================================================

===============================================================================
PASSO 5: ENVIAR OS ARQUIVOS PARA O CLAUDE
===============================================================================

Logo após a primeira mensagem, envie os arquivos:

No Claude Code:
- Use o comando "@" e selecione os arquivos
- Ou arraste e solte os arquivos na conversa

Arquivos para enviar (em ordem):
1. RESUMO_TECNICO.txt
2. TUTORIAL_ATUALIZACAO_MENSAL.md
3. CHECKLIST_ATUALIZACAO.md
4. modulo_fechamento_mes.py (se tiver)
5. modulo_fluxo_caixa.py (se tiver)

===============================================================================
PASSO 6: SEGUIR AS INSTRUÇÕES DO CLAUDE
===============================================================================

O Claude vai te guiar através do processo. Perguntas típicas:

"Você já tem o repositório clonado?"
→ Responda: Sim/Não

"Você tem Git instalado?"
→ Responda: Sim/Não

"Qual o caminho da pasta do projeto?"
→ Responda: ex: C:\dash-financeiro ou ~/dash-financeiro

"Você já tem Personal Access Token?"
→ Responda: Sim/Não (se não tiver, ele vai te ajudar a criar)

===============================================================================
PASSO 7: COMANDOS QUE O CLAUDE VAI SUGERIR
===============================================================================

O Claude provavelmente vai te dar comandos como estes.
Você só precisa copiar e colar no terminal do notebook:

# Clonar (primeira vez)
git clone https://github.com/Kalarien/dash-financeiro.git
cd dash-financeiro

# Atualizar (se já tem)
cd dash-financeiro
git pull origin main

# Depois que você editar os arquivos (o Claude vai te ajudar nisso):
git add modulo_fechamento_mes.py modulo_fluxo_caixa.py
git commit -m "Atualização dados financeiros - Outubro 2025"
git push origin main

===============================================================================
DICAS IMPORTANTES
===============================================================================

✅ FAÇA:
- Tenha todos os dados coletados ANTES de começar
- Use o CHECKLIST para não esquecer nada
- Teste localmente antes de fazer push (se possível)
- Mantenha o Personal Access Token em lugar seguro

❌ NÃO FAÇA:
- Não compartilhe seu Personal Access Token
- Não faça push sem ter certeza dos dados
- Não edite arquivos que o Claude não mencionou
- Não use git push --force (a menos que o Claude explicitamente diga)

===============================================================================
SE ALGO DER ERRADO
===============================================================================

1. MANTENHA A CALMA! É possível reverter mudanças no Git

2. Diga ao Claude exatamente qual erro apareceu

3. O Claude pode te ajudar a:
   - Reverter commits: git revert HEAD
   - Desfazer mudanças locais: git checkout -- arquivo.py
   - Ver o que mudou: git diff

4. Se o dashboard ficar quebrado:
   - Acesse https://share.streamlit.io/
   - Clique nos 3 pontinhos do seu app
   - Clique em "Reboot app"
